Cómo usar esta plataforma

🚑 Estado de unidades — Muestra si cada unidad está lista para salir. Verde = lista, amarillo = faltantes menores, rojo = crítica.


Revisar una unidad — Haz clic en los artículos del checklist para marcarlos como presentes. Luego pulsa "Guardar revisión" para registrar en el historial.


Registrar revisión general — Guarda el estado actual de todas las unidades en el historial de una sola vez.


📋 Historial — Registro completo de todas las revisiones con fecha, responsable y estado.


➕ Nueva unidad — Agrega una nueva unidad con su propio checklist personalizado o usa las plantillas predefinidas.


💾 Los datos se guardan en este navegador.
