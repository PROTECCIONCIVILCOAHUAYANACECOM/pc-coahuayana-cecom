Cómo usar este sistema

📦 Inventario — Agrega, edita y consulta todos los artículos. Haz clic en cualquier valor para editarlo. La columna "Alerta" se actualiza automáticamente.


🔄 Movimientos — Registra cada entrada o salida de material. El inventario se actualiza automáticamente al guardar un movimiento.


📊 Resumen — Vista rápida de totales y artículos con alerta.


💾 Los datos se guardan en este navegador. Para compartir con tu equipo, abre el archivo HTML en cada computadora o celular. Para una versión en la nube con múltiples usuarios simultáneos, se necesita un servidor web.


⬇ Exportar CSV — Descarga el inventario como archivo CSV que puedes abrir en Excel o Google Sheets.
